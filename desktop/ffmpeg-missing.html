ffmpeg est introuvable

Flix a besoin de ffmpeg et ffprobe (version 6 ou plus récente) installés sur cette machine pour analyser et lire tes vidéos. Flix ne les télécharge jamais automatiquement — installe-les toi-même avec le gestionnaire de paquets de ton système, puis clique sur « Réessayer ».

Debian / Ubuntu
sudo apt install ffmpeg
Fedora
sudo dnf install ffmpeg
Arch Linux
sudo pacman -S ffmpeg
Windows
winget install ffmpeg

Si ffmpeg est déjà installé mais reste introuvable, vérifie qu'il est bien dans le PATH, ou définis les variables d'environnement FFMPEG_PATH et FFPROBE_PATH pointant vers les binaires.
